Linear Regression on California Housing Dataset
Project Overview
This project demonstrates an end-to-end Machine Learning workflow using Linear Regression to predict median house values based on socio-economic and geographic features from the California Housing dataset. It includes data loading, exploratory data analysis (EDA), model training, and evaluation.

Dataset Description
The California Housing dataset is available in the scikit-learn library.

Instances: 20,640
Features: 8
Target: Median House Value (MedHouseVal)
Features
Median Income
House Age
Average Rooms
Average Bedrooms
Population
Average Occupancy
Latitude
Longitude
